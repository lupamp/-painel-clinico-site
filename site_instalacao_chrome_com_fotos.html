INSTALAÇÃO MANUAL NO GOOGLE CHROME

Como instalar a extensão no Google Chrome

Baixe o arquivo ZIP da extensão, extraia a pasta e siga o passo a passo abaixo para instalar manualmente no Chrome usando o modo do desenvolvedor.

⬇ Baixar arquivo ZIP da extensão Ver passo a passo
Importante: para o botão funcionar no GitHub, envie também o arquivo Linha cuidado.zip para o mesmo repositório/pasta onde estiver este arquivo HTML.

Antes de começar

Você vai precisar de um computador com Google Chrome instalado. O Chrome não instala o ZIP diretamente; antes, é necessário extrair o conteúdo do arquivo para uma pasta normal.

  • Baixe o arquivo ZIP da extensão.
  • Extraia o ZIP para uma pasta.
  • Abra a página chrome://extensions.
  • Ative o Modo do desenvolvedor.
  • Clique em Carregar sem compactação.
  • Selecione a pasta extraída.

Download rápido

Se o botão de download não funcionar, verifique se o arquivo ZIP foi enviado para o GitHub com o nome exato Linha cuidado.zip.

Compatibilidade

Esse método funciona no Google Chrome em Windows. Em navegadores baseados em Chromium, o fluxo é semelhante.

Dica

Depois da instalação, você pode fixar a extensão no Chrome para acessar mais facilmente no canto superior direito.

Passo a passo com imagens

A sequência abaixo mostra exatamente o fluxo recomendado para instalar a extensão de forma manual.

1
DOWNLOAD

Baixe o arquivo ZIP

Clique no botão Baixar arquivo ZIP da extensão. O arquivo será salvo no seu computador.

  • Se estiver no GitHub Pages, o link abrirá o arquivo enviado junto com o site.
  • Se nada acontecer, confirme se o ZIP está no mesmo local do site.
2
EXTRAÇÃO

Extraia o conteúdo do ZIP

Após o download, clique com o botão direito no arquivo ZIP e escolha Extrair tudo ou opção semelhante.

  • Você deve terminar com uma pasta da extensão, não com o arquivo ZIP.
  • É essa pasta que será selecionada no Chrome.
3
CHROME

Abra a página de extensões do Chrome

No navegador, digite chrome://extensions na barra de endereços e pressione Enter.

  • Essa é a área onde o Chrome gerencia as extensões instaladas.
4
MODO DO DESENVOLVEDOR

Ative o modo do desenvolvedor

No canto superior direito da página de extensões, ative a chavinha Modo do desenvolvedor.

  • Sem isso, o botão para carregar a pasta da extensão não aparece.
5
SELEÇÃO DA PASTA

Clique em “Carregar sem compactação”

Depois de ativar o modo do desenvolvedor, clique no botão Carregar sem compactação.

  • Uma janela do Windows será aberta para você escolher a pasta da extensão.
6
FINALIZAÇÃO

Selecione a pasta extraída da extensão

Escolha a pasta que você extraiu do ZIP. Assim que selecionar, o Chrome instalará a extensão.

  • Se aparecer algum erro, confira se você selecionou a pasta correta.
  • Normalmente ela deve conter o arquivo manifest.json.
Atenção O Google Chrome não instala uma extensão diretamente de um arquivo ZIP em site comum. O procedimento correto é: baixar o ZIP, extrair a pasta e usar Carregar sem compactação na página de extensões.

Perguntas frequentes

Preciso manter a pasta extraída no computador?

Sim. Como a instalação foi feita com “Carregar sem compactação”, o Chrome utiliza aquela pasta para manter a extensão funcionando.

Posso instalar clicando direto no ZIP?

Não. Primeiro você precisa extrair o ZIP para uma pasta comum. Depois, selecione essa pasta no Chrome.

Se der erro na instalação, o que devo verificar?

Confirme se você selecionou a pasta certa e se dentro dela existe um arquivo manifest.json. Sem ele, o Chrome não consegue reconhecer a extensão.
